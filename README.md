# Formulario React Python

Aplicación web que integra React en el frontend y Python en el backend, conectada a una base de datos MySQL. Diseñada para reforzar conocimientos de programación, permite a los usuarios gestionar datos a través de un formulario dinámico, facilitando la inserción, consulta, actualización y eliminación de información.
